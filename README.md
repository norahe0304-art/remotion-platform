# remotion-platform

Reusable Remotion platform for Nora projects.

## Workspace modules

- `packages/remotion-controls`: Shared props schema + VO timeline helpers + duration normalization
- `packages/remotion-styles`: Reusable branding and motion style tokens/components
- `templates/remotion-template-nora`: Starter project wired to shared packages
- `prompts`: Curated prompt library with attribution metadata

## Quick start

```bash
npm install
npm run prompts:list
```
